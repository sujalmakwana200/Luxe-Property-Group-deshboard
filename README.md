## 📌 Project Overview
The **Luxe Property Group Sales Dashboard** is an interactive, data-driven web application designed for real estate brokers, agency owners, and business stakeholders. It provides real-time geographic pricing analysis, market trend visualization, and top-level KPI tracking to help drive strategic, data-backed decisions in the premium housing market.

## ✨ Key Features
* **Dynamic KPI Tracking:** Instantly calculates Total Sales Volume, Average Sale Price, and Days on Market based on active filters.
* **Interactive Geographic Mapping:** Utilizes `Plotly Mapbox` to plot property sales coordinates, allowing stakeholders to visually analyze neighborhood pricing trends.
* **Custom File Uploading:** Users can seamlessly drag and drop their own daily or weekly sales CSV files directly into the sidebar to visualize fresh data without touching a single line of code.
* **Executive Filtering:** Real-time multi-select and slider widgets to drill down into specific price ranges and property types (Villas, Penthouses, etc.).

## 💻 Tech Stack
* **Language:** Python
* **Web Framework:** Streamlit
* **Data Processing:** Pandas, NumPy
* **Data Visualization:** Plotly Express
